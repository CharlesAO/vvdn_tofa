 II.    Tools Documents
 ➤    moviTools Documents
Section Description
1. moviTools Release Notes General tools re-factoring due to requirements for Myriad 2 support.
2. moviTools Introduction Movidius Tools are a set of tools used to build, run and debug applications for Movidius Myriad processors.
3. moviAsm The Movidius Assembler ( moviAsm ) is the Software Component which is responsible for producing output binary files in the specified format, according to the latest ISA specifications.
4. moviCompile moviCompile is Movidius' C/C++ compiler, a tool which compiles C and C++ code. moviCompile is based on CLang and the LLVM infrastructure. moviCompile is the default compiler used to compile code for Movidius' SHAVE core.
5. moviConvert The moviConvert tool converts the mof file into: hex files used for vcs simulation; mvcmd file used as boot image; mvupd file used as update boot image; .elf file used as output cross-checking.
6. moviDebug 2 Version 2 of the Movidius Debugger (moviDebug), an application that is used to test and debug Movidius code on different platforms.
7. moviDebug 2 features This document presents the features of the Movidius Debugger 2 (moviDebug2).
8. moviDebugServer Movidius Debug Server ( moviDebugServer ) is an application that is used to provide the link between the moviDebug (Movidius Debugger) and the hardware targets ( ASIC or FPGA ).
9. moviDump The Movidius Object Dumper ( moviDump ) is capable of dumping an object file in a readable format.
10. moviEclipse features This document presents the features of moviEclipse.
11. moviSim MoviSim is a configurable simulator designed for heterogeneous multiprocessor systems.
12. moviProf Movidius Profiler (moviProf) is an application that is used to decode profiled functions instrumented with the functionProfiler MDK component. It is integrated in Eclipse and can be used from the command line as well.
